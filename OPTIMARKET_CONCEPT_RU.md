# OPTIMARKET — Концепт и архитектура MVP (RU)

> **Цель:** создать интеллектуальную платформу выбора товаров в формате
> **Premium / Optimum / Economy** без склада и логистики, с монетизацией
> через партнёрские ссылки и объяснениями от LLM.

## 1. Краткое описание продукта

**OPTIMARKET** — не магазин, а **ИИ‑куратор выбора**, который:

- собирает реальные данные о товарах (характеристики, цены, отзывы);
- объясняет различия между моделями;
- показывает 3 уровня выбора: **Premium / Optimum / Economy**;
- переводит пользователя на внешнюю покупку через affiliate‑ссылки.

**Ключевая идея:** пользователь получает прозрачный, экспертный и краткий выбор,
а не каталог на 1000 позиций.

---

## 2. Ниша MVP + альтернативные ниши

### Базовая ниша (MVP)
**Инструменты и электроника для дома / мастеров (DIY, ремонт, СТО)**

**Почему:**
- легко структурировать характеристики;
- понятные уровни выбора;
- много партнёрских офферов в Amazon/Awin/Admitad/Heureka;
- постоянный спрос (B2B + B2C).

### Дополнительные ниши для проверки
1. **Умный дом и безопасность**
   - камеры, датчики, умные замки, сигнализации.
2. **Кухонная техника**
   - кофемашины, мультиварки, блендеры: параметризация простая.
3. **Авто‑аксессуары**
   - видеорегистраторы, компрессоры, зарядные устройства.
4. **Красота и здоровье**
   - массажёры, ирригаторы, триммеры, уходовые гаджеты.

## 4. Архитектура (MVP 1.0)

**Слои:**
1. **Data Layer** — каталог и партнёрские API.
2. **AI Layer** — подбор, объяснения, анализ отзывов.
3. **Interface Layer** — сайт, бот, API.

### Data Layer
- Awin / Admitad / Amazon / Heureka API.
- JSON‑каталоги (автообновление 24 ч).
- Perplexity: отзывы, тесты, экспертные обзоры.

### AI Layer
- GPT‑5: подбор и объяснение выбора.
- Perplexity: поиск и summarization отзывов.
- Ранжирование: скоринг «цена × качество × надёжность».

### Interface Layer
- Web (Next.js/Tilda/Webflow) или Telegram Mini App.
- Карточки Premium/Optimum/Economy + CTA «Сравнить / Купить / Сохранить».

---

## 5. LLM‑модуль и Perplexity Engine

### Роль LLM
- уточняет запрос и контекст;
- подбирает 3 уровня;
- объясняет различия;
- предупреждает о слабых местах (из отзывов).

### Perplexity Review Intelligence Engine (PRIE)
- собирает отзывы и тесты;
- извлекает плюсы/минусы;
- даёт краткую экспертную сводку.

---

## 6. MVP Roadmap (30–60 дней)

1. **Регистрация в партнёрках** (Awin, Admitad, Amazon, Heureka).
2. **Сбор 10–15 товаров** по основной категории.
3. **Прототип витрины** (Tilda/Notion/Next.js).
4. **Интеграция GPT‑5 + Perplexity** для объяснений и отзывов.
5. **Запуск Telegram‑бота** как второй точки входа.
6. **Тестовый трафик** (Reddit/YouTube/Telegram).

## 8. UX/дизайн‑принципы

- минимализм, доверие, «инженерный тон»;
- 3 карточки = 3 решения, без перегруза;
- спокойные CTA: «Подробнее», «Сравнить», «Купить»;
- упор на объяснение и источники, а не маркетинг.
